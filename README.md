# Netflix Movies & TV Shows Data Analysis

## Project Overview

This project is an **Exploratory Data Analysis (EDA)** of the Netflix Movies and TV Shows dataset using **Python**, **Pandas**, and **Matplotlib**.

The objective of this project is to analyze Netflix's content library, identify trends, and generate meaningful business insights through data visualization.

This project is part of my **Data Analyst Portfolio** and demonstrates my ability to clean, explore, analyze, and visualize real-world datasets.

---

## Project Objectives

- Load and explore the Netflix dataset
- Perform data cleaning and identify missing values
- Analyze Movies vs TV Shows distribution
- Identify the top countries producing Netflix content
- Analyze content ratings
- Identify the most common genres
- Analyze Netflix content growth over the years
- Generate business insights using visualizations

---
